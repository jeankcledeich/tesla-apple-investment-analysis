# Tesla-Apple Investment Analysis 2024

## Introduction
This repository contains the foundation for a semester‑long investment research project comparing **Tesla (TSLA)** and **Apple (AAPL)** for the year 2024. The analysis will cover:
- Comprehensive financial metrics and historic price data
- Income statement trends
- Institutional ownership and analyst sentiment for Tesla
- Market sentiment extracted from recent news headlines
- Technical analysis (RSI, MACD) – to be added in a later phase

The data files and summaries are stored in this repo and a local backup directory has been created for reference.
